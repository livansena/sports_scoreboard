# Sports Scoreboard

Modern sports dashboard built with Ruby on Rails.

## Features

- Team management
- Match management
- Live-style scoreboard UI
- Dark dashboard interface
- PostgreSQL database
- Responsive layout

## Tech Stack

- Ruby on Rails
- PostgreSQL
- Tailwind CSS
- JavaScript

```bash
bundle install
bin/rails db:create
bin/rails db:migrate
bin/rails server
```
